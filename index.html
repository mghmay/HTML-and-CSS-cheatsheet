CSS Specificity and Selectors

Selectors

Selector (in order from most to least specific) Selector weight Description
Inline styles 1000 These are styles which are placed directly in the html body. Using the style attribute i.e. style="".
IDs 100 These elements are selected using the # i.e. #navbar
Classes, psuedo-classes, attribute selectors 10 Classes are selected using the .. i.e. .bold. Psuedo-classes also fall under this catagory i.e. :hover. As do attribute selectors using square brackets i.e. [href].
Elements and psuedo-elements 1 These elements are selected using the their HTML code i.e. h1. Psuedo elements also fall under this catagory i.e. :before.
Universal selector 0 This is usually declared at the beginning of a document using *.

For more information on CSS selectors. See this reference guide: CSS Selector Reference

Specificity

Start at 0, add 100 for each ID value, add 10 for each class value (or pseudo-class or attribute selector), add 1 for each element selector or pseudo-element. The higher value wins the style.

Examples

Selector Specificity calculation What is selected
p 1 All paragraphs in the body
p.test 1 + 10 = 11 All paragraphs with the class of 'test'
p#demo 1 + 100 = 101 All paragraphs with the ID of 'demo'.
<p style="color: pink;"> 1000 The specific paragraph with the style declatation.
#demo 100 The element with this ID.
.test 10 The element with this class.
p.test1.test2 1 + 10 + 10 = 21 All paragraphs with both the test1 and test 2 classes.
