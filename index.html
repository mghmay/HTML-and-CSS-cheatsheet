CSS Specificity and Selectors

Selector (in order from most to least specific) Selector weight Description
Inline styles 1000 These are styles which are placed directly in the html body. Usind the style attribute i.e. style=""
IDs 100 These elements are selected using the '#' i.e. #navbar
Classes, psuedo-classes, attribute selectors 10 Classes are selected using the '.' i.e. .bold. Psuedo-classes also fall under this catagory i.e. :hover. As do attribute selectors using square brackets i.e. [href].
Elements and psuedo-elements 1 These elements are selected using the their HTML code i.e. h1. Psuedo elements also fall under this catagory i.e. :before.

For more information on CSS selectors. See this reference guide: CSS Selector Reference
